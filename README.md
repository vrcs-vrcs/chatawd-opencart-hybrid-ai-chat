# ChatAWD - AI Chat Assistant & Abandoned Cart Recovery for OpenCart

Boost your OpenCart store sales with **ChatAWD**, the ultimate hybrid customer engagement and conversion optimization system. This module combines advanced artificial intelligence with real-time human dispatcher takeover, automated email marketing, and interactive gamification to maximize your revenue.

Fully compatible with **OpenCart 3.x** and **OpenCart 4.x**.

---

## 🚀 Key Features to Boost Your Sales

### 🤖 Hybrid AI Chat Assistant
* **Smart AI Support:** Automated, 24/7 customer service using OpenAI (GPT-4o-mini) and Google Gemini APIs.
* **Instant Live Agent Takeover:** If a customer needs human assistance, a live dispatcher can step in instantly using the dedicated CRM dashboard.
* **Context Synchronization:** The dispatcher can see the customer's current cart contents in real-time, reconstruct the cart, and push custom offers to close the sale.

### 📧 5-Step Abandoned Cart Email Recovery
* **Automated Autopilot:** Automatically send up to 5 fully customizable, timed email reminders to customers who left items in their cart.
* **Time Delay Controls:** Fine-tune delays (hours or days) and message bodies for each step.
* **Direct Conversion Impact:** Bring back lost visitors and recover abandoned revenue automatically.

### 🎡 Gamified Loyalty Tools (Wheel of Fortune)
* **Interactive Engagement:** Display a high-converting Wheel of Fortune inside the chat widget.
* **Email Capture:** Encourage visitors to spin the wheel in exchange for discount coupons, growing your newsletter list.

## ⚙️ Installation Guide

### For OpenCart 4.x
1. Download the `chatawd_4.ocmod.zip` file.
2. Go to your OpenCart Admin -> **Extensions** -> **Installer**.
3. Upload the zip file.
4. Go to **Extensions** -> **Extensions** -> Select **Modules** -> Install **ChatAWD AI Chat**.
5. Edit the settings, register on [vrcs.hu](https://www.vrcs.hu/) to get your API keys, and start boosting your sales.

### For OpenCart 3.x
1. Download the `chatawd_3.ocmod.zip` file.
2. Go to your OpenCart Admin -> **Extensions** -> **Installer**.
3. Upload the zip file.
4. Go to **Extensions** -> **Modifications** -> Click the **Refresh** button in the top right.
5. Go to **Extensions** -> **Extensions** -> Select **Modules** -> Install **ChatAWD AI Chat** and configure your settings.
